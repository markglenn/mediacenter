mediacenter

mediacenter is a vagrant setup to build a media center server.

The server contains everything needed to manage automatic downloads.

Includes the following services:

  • SABnzbd
  • Sick Beard
  • Couch Potato
  • Headphones

The server also includes Plex with enough power to handle 4 streams simultaneously. This by default sets up two cores and two gigabytes of RAM. This is based on recommendations by Plex.

  • Plex

Requirements

  • Vagrant
  • Virtualbox

Quickstart

Use this to quickly create the vagrant box.

git clone https://github.com/markglenn/mediacenter.git
cd mediacenter
vagrant up
